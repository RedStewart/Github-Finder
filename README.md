# Github Finder

A ReactJS app created by following the 'React Front To Back 2019' Udemy course lectured by Brad Traversy. Ability to search for a Github user and view information about their profile and their 5 most recent repos.
The app implements state management with the ReactJS Context API (useReducer, useContext, useState, and useEffect) to avoid prop drilling and as an alternative to Redux.
